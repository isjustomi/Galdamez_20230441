¿Que son los componentes pasivos de la red?

Los componentes pasivos en una red son aquellos que no requieren energía eléctrica para funcionar y no pueden amplificar o generar señales. Su función principal es conectar, organizar y proteger los cables y dispositivos de la red.

Descripcion de cada componente Pasivo

RJ45

El conector RJ45 es un tipo de conector utilizado comúnmente para redes Ethernet. Tiene ocho pines y se utiliza para conectar cables de par trenzado a dispositivos de red como computadoras, routers y switches.

Jacks o Keystone

Los jacks o keystone son conectores modulares que se instalan en placas de pared, paneles de parcheo o cajas de montaje en superficie. Proporcionan un punto de terminación para los cables Ethernet y permiten una conexión fácil y organizada.

Cable UTP

El cable UTP (Unshielded Twisted Pair) es un tipo de cable de cobre que consiste en pares de hilos trenzados sin blindaje. Es ampliamente utilizado en redes de comunicación para la transmisión de datos debido a su flexibilidad y bajo costo.

Patch Panel

Un patch panel es un dispositivo de hardware con múltiples puertos que organiza y conecta cables de red. Facilita la gestión del cableado estructurado y permite la interconexión de cables de red con equipos activos como switches y routers.

Patch Cord

El patch cord es un cable de red de corta distancia con conectores RJ45 en ambos extremos. Se utiliza para conectar dispositivos de red a un patch panel o directamente a un switch o router.

Fibra Óptica

La fibra óptica es un medio de transmisión que utiliza pulsos de luz para transmitir datos a altas velocidades y largas distancias. Es ideal para aplicaciones que requieren un ancho de banda elevado y una baja latencia.

Canaletas

Las canaletas son conductos utilizados para proteger y organizar cables eléctricos y de datos. Ayudan a mantener el cableado ordenado y protegido de daños físicos.

¿Que son los componentes activos de la red?

Los componentes activos de una red son aquellos que requieren energía para funcionar y que tienen la capacidad de amplificar, modificar, conmutar o procesar las señales de datos.

Descripcion de cada componente activo

Switch

Un switch es un dispositivo de red que conecta múltiples dispositivos dentro de una misma red local (LAN). Funciona en la capa 2 del modelo OSI, utilizando direcciones MAC para enviar datos al dispositivo correcto. Los switches pueden tener múltiples puertos, permitiendo la conexión de varios dispositivos simultáneamente.

Router

Un router es un dispositivo que dirige el tráfico de datos entre diferentes redes. Funciona en la capa 3 del modelo OSI, utilizando direcciones IP para determinar la ruta más eficiente para enviar datos. Los routers permiten la comunicación entre redes locales y la red global de Internet.

Tarjeta de red

Una tarjeta de red (NIC) es un componente de hardware que permite a un dispositivo conectarse a una red. Puede ser una tarjeta interna o un adaptador externo, y se utiliza para la transmisión y recepción de datos a través de la red.

Interruptor inteligente WiFi

Un interruptor inteligente WiFi es un dispositivo que permite controlar la iluminación y otros aparatos eléctricos de forma remota a través de una conexión WiFi. Se puede programar y controlar mediante aplicaciones móviles o asistentes de voz como Alexa y Google Home.

Access Point

Un punto de acceso es un dispositivo que permite a los dispositivos inalámbricos conectarse a una red cableada. Extiende la cobertura de la red WiFi, proporcionando una conexión estable y de alta velocidad en áreas donde la señal del router principal no llega adecuadamente.

Herramientas de cableado

Ponchadora

La ponchadora es una herramienta utilizada para insertar los cables en los conectores modulares, como los conectores RJ45. Es esencial para asegurar una conexión firme y estable.

Crimpadora

La crimpadora se utiliza para unir un conector al cable, asegurando que los contactos metálicos del conector perforen el aislamiento del cable y hagan contacto con los conductores internos. Es fundamental para la creación de cables de red personalizados.

Desforradora

La desforradora se utiliza para quitar el aislamiento exterior de los cables sin dañar los conductores internos. Es una herramienta precisa que facilita el trabajo de preparación del cable.

Tester/Probador de cables

El tester o probador de cables se utiliza para verificar la continuidad y la correcta conexión de los cables. Permite identificar problemas como cortocircuitos, cables cruzados o conexiones sueltas, asegurando que el cableado esté funcionando correctamente.

Tipos de cableado

Cableado horizontal

El cableado horizontal es la parte del cableado estructurado que conecta las salidas de telecomunicaciones en el área de trabajo con los paneles de parcheo en la sala de telecomunicaciones. Este tipo de cableado se extiende a lo largo de una misma planta del edificio y suele utilizar cables UTP, STP o fibra óptica.

(las lineas verdes)

Cableado vertical o backbone

El cableado vertical o backbone es el sistema de cableado que conecta los diferentes pisos de un edificio. Este tipo de cableado se utiliza para interconectar los racks de telecomunicaciones principales y secundarios, permitiendo la transmisión de datos entre diferentes niveles del edificio. Generalmente, se utiliza fibra óptica o cables de cobre para estas conexiones.

(las lineas rojas)

Tipos de cableado horizontales

Cables de par trenzado sin blindar

El cable UTP es el más común en instalaciones de redes. Consiste en cuatro pares de hilos de cobre trenzados para reducir la interferencia electromagnética. Es económico y fácil de instalar, ideal para redes de área local (LAN).

Cables de par trenzado blindados

El cable STP tiene una capa adicional de blindaje que protege los pares trenzados de interferencias externas. Es más caro y rígido que el UTP, pero ofrece mejor rendimiento en entornos con alta interferencia electromagnética.

Cables de fibra óptica multimodo

Este tipo de cable utiliza fibras de vidrio o plástico para transmitir datos mediante pulsos de luz. Es ideal para distancias más largas y ofrece alta velocidad de transmisión. La fibra multimodo es adecuada para aplicaciones dentro de edificios.

Normativas para cables de red

Las normas T568A y T568B son los dos estándares principales para el cableado de red Ethernet, específicamente para la terminación de cables en conectores RJ-45.

¿Para que se utiliza un cable directo en la red?

Un cable directo, también conocido como cable de parcheo o cable de conexión, se utiliza para conectar dispositivos diferentes dentro de una red.

¿Para que se utiliza un cable cruzado en la red?

Un cable cruzado se utiliza para conectar dispositivos similares directamente entre sí sin necesidad de un switch o hub intermedio.
